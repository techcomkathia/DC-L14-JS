Tratamento de Erros com try...catch no JavaScript

O try...catch é utilizado para evitar que erros interrompam a execução do programa. Ele permite capturar erros e definir uma ação personalizada quando algo dá errado.

Estrutura Básica

            try {
                // código que pode gerar erro
            } catch (erro) {
                // tratamento do erro
            }

Exemplo

            try {
                let resultado = numero + 10
                console.log(resultado)
            } catch (erro) {
                console.log("Ocorreu um erro!")
            }

No exemplo acima, a variável numero não existe, então o JavaScript gera um erro. O catch captura esse erro e exibe uma mensagem no console.

Objeto de Erro

O parâmetro do catch contém informações sobre o erro ocorrido. 

            try {
                console.log(usuario.nome)
            } catch (erro) {
                console.log(erro.message)
            }

finally

O bloco finally é executado sempre, independentemente de ocorrer erro ou não. 

            try {
                console.log("Executando...")
            } catch (erro) {
                console.log("Erro encontrado")
            } finally {
                console.log("Fim do processo")
            }

Funções Async e Await no JavaScript

As palavras-chave async e await são utilizadas para trabalhar com operações assíncronas de forma mais simples e organizada.

Função Async

Uma função marcada com async sempre retorna uma Promise. 

            async function mensagem() {
                return "Olá Mundo!"
            }

Await

O await faz o JavaScript esperar o resultado de uma operação assíncrona antes de continuar. Ele só pode ser utilizado dentro de funções async. 

            async function carregarDados() {
                let resposta = await fetch("https://api.exemplo.com")

                console.log(resposta)
            }

Exemplo Completo

            async function buscarUsuario() {
                try {
                    let resposta = await fetch("https://jsonplaceholder.typicode.com/users/1")

                    let usuario = await resposta.json()

                    console.log(usuario)
                } catch (erro) {
                    console.log("Erro ao buscar usuário")
                }
            }

            buscarUsuario()

Nesse exemplo, o await aguarda a resposta da API e a conversão dos dados para JSON. O try...catch é utilizado para tratar possíveis erros durante a requisição.
